# Advent Of Code 2024

### Instructions
1. `pnpm install`
2. Pick the day you want to run.
3. Add your input into a file named `./src/dayX/dayX-input.txt` file.
4. Run with `pnpm ts-node ./src/day6/day6-2.ts`
